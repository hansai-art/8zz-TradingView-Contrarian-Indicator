## 概述

「8zz 反指標」是一個 TradingView Pine Script v6 的 overlay 指標，記錄特定交易者（8zz/Banini）的公開買賣操作，並以**反向信號**的角度加以呈現。核心邏輯：當該交易者做多，指標顯示偏多（反指）；做空/買進時顯示偏空（反指）。搭配勝率統計面板，讓觀察者評估操作翻轉後的市場走向。

---

## 指標參數

| 參數 | 類型 | 預設值 | 範圍 | 說明 |
|------|------|--------|------|------|
| `offset_pct` | float | 2.0 | 0.5 ~ 5.0 (步進 0.1) | 箭頭距 K 棒高低點的距離（以高低價差 % 計） |
| `showDots` | bool | false | — | 開啟後，同方向重複訊號顯示小圓點（加倉圓點） |
| `showPanel` | bool | true | — | 顯示右上角資訊面板 |
| `holdBars` | int | 5 | 1 ~ 50 | 翻轉後觀察幾根 K 棒後結算勝敗（1H線 5 = 5小時；日線 5 = 一週） |

## 事件資料格式

事件在 `barstate.isfirst` 時一次性寫入三個平行陣列：

| 陣列 | 類型 | 說明 |
|------|------|------|
| `evt_time` | `int[]` | Unix 毫秒時間戳 |
| `evt_dir` | `int[]` | 方向：`1` = 偏多（停損/賣出）、`-1` = 偏空（買進/抄底） |
| `evt_tips` | `string[]` | Tooltip 文字（含操作說明、指標方向、FB 貼文時間） |

目前收錄 **33 筆**事件，時間範圍 2025/02/09 ~ 2025/04/10。

### 方向定義

| `evt_dir` | 代表行為 | 指標顯示 | 顏色 | 符號 |
|-----------|----------|----------|------|------|
| `1` | 停損賣出 / 減碼 | 偏多 ▲ | `#00E676`（綠） | `▲` label_up（K 棒下方） |
| `-1` | 買進 / 抄底 / 加碼 | 偏空 ▼ | `#FF1744`（紅） | `▼` label_down（K 棒上方） |

## 主邏輯流程

```
每根 K 棒：
  1. 結算判斷
     - 若距上次翻轉已過 holdBars 根 K 棒
     - 計算 PnL（多單：close - flipPrice；空單：flipPrice - close）
     - PnL > 0 → wins++；否則 losses++

  2. 事件掃描（遍歷所有事件）
     - 條件：time >= evt_time[i] AND time[1] < evt_time[i]（精確落在該 K 棒）
     - 若方向翻轉（d != currentDir）：
         * 更新 currentDir、posCount=1、記錄 flipBar/flipPrice/flipDir
         * totalFlips++
         * 繪製大箭頭 label（▲ 或 ▼）
     - 若方向相同（加碼）：
         * posCount = min(posCount + 1, 5)
         * showDots 開啟時繪製小圓點 label
```

### 狀態變數

| 變數 | 說明 |
|------|------|
| `currentDir` | 目前持倉方向（0=無、1=多、-1=空） |
| `posCount` | 目前倉位格數（1~5） |
| `flipBar` | 最近一次翻轉的 bar_index |
| `flipPrice` | 翻轉當根 K 棒的收盤價（作為勝敗基準） |
| `flipDir` | 翻轉時的方向 |
| `wins / losses` | 累計勝敗場次 |
| `totalFlips` | 累計翻轉次數 |

## 資訊面板

位置：右上角（`position.top_right`），2 欄 × 5 列。

| 列 | 欄 0（標籤） | 欄 1（數值）| 數值顏色規則 |
|----|------------|------------|------------|
| 0 | `8zz 反指標` | 偏多 ▲ / 偏空 ▼ / 等訊號 | 多=綠、空=紅、無=灰 |
| 1 | 倉位 | `N/5 (N*20%)` | 白色 |
| 2 | 勝率 | `XX.X% (NW NL)` | ≥60%=綠、≥40%=黃、<40%=紅 |
| 3 | 觀察窗 | `N bars` | 灰色 |
| 4 | 翻轉次數 | `N` | 灰色 |

## Label 樣式

| 類型 | `style` | `size` | `color`（背景） | `textcolor` |
|------|---------|--------|----------------|-------------|
| 多頭箭頭 ▲ | `label.style_label_up` | `size.huge` | 透明黑 | `#00E676` |
| 空頭箭頭 ▼ | `label.style_label_down` | `size.huge` | 透明黑 | `#FF1744` |
| 加倉圓點 | `label.style_circle` | `size.tiny` | 綠/紅實色 | — |

箭頭 Y 座標：
- 多頭：`low - (high - low) * offset_pct`
- 空頭：`high + (high - low) * offset_pct`

圓點 Y 座標偏移縮小為 `offset_pct * 0.3`。

`max_labels_count=500`；`timeframe=""`, `timeframe_gaps=false`（跟隨圖表時間框架）。

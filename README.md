## 簡介

「8zz 反指標」是一個 TradingView **Pine Script v6** overlay 指標。

它記錄了特定交易者（8zz/Banini）在 Facebook 公開發文的買賣操作，並以**反向信號**呈現在 K 線圖上。核心哲學：**他停損賣出 → 指標偏多 ▲；他買進抄底 → 指標偏空 ▼。**

內建簡易回測：翻轉後 N 根 K 棒結算勝敗，右上角面板即時顯示勝率。

---

## 功能特色

| 功能 | 說明 |
|------|------|
| **方向箭頭** | 翻轉時於 K 棒上/下方顯示大型 ▲ / ▼ 標籤 |
| **加倉圓點** | 同方向重複訊號可選擇顯示小圓點（可開關） |
| **Tooltip** | 懸停箭頭可查看原始貼文摘要與時間 |
| **資訊面板** | 右上角顯示當前方向、倉位、勝率、翻轉次數 |
| **簡易回測** | 可設定觀察 K 棒數，自動統計勝敗 |

---

## 事件資料

目前收錄 **33 筆**公開操作紀錄，時間範圍：**2025/02/09 ~ 2025/04/10**

| 方向 | 代表行為 | 顏色 |
|------|----------|------|
| ▲ 偏多 | 停損賣出 / 減碼 / 出清 | 綠 `#00E676` |
| ▼ 偏空 | 買進 / 抄底 / 加碼 | 紅 `#FF1744` |

## 參數設定

### 顯示

| 參數 | 預設 | 說明 |
|------|------|------|
| 箭頭距離 % | `2.0` | 箭頭與 K 棒高低點的距離（0.5 ~ 5.0） |
| 顯示加倉圓點 | `false` | 同方向訊號顯示小圓點 |
| 顯示資訊面板 | `true` | 右上角統計面板 |

### 回測

| 參數 | 預設 | 說明 |
|------|------|------|
| 觀察K棒數 | `5` | 翻轉後幾根 K 棒結算勝敗（1H線: 5=5小時；日線: 5=一週） |

## 安裝方式

1. 開啟 [TradingView](https://www.tradingview.com) 並進入 Pine Script 編輯器
2. 複製 [`8zz-indicator.pine`](./8zz-indicator.pine) 的全部內容
3. 貼入編輯器並點擊「儲存」→「加到圖表」

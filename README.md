## 這是什麼？

「8zz 反指標」是一個 TradingView K 線圖指標，記錄了巴逆逆（8zz）在 Facebook 公開發文的買賣操作，並以**反向信號**顯示在圖表上。

核心邏輯很簡單：
- 她停損賣出 → 指標顯示**偏多 ▲**（綠色箭頭）
- 她買進抄底 → 指標顯示**偏空 ▼**（紅色箭頭）

收錄 **63 筆** FB 貼文操作紀錄，時間橫跨 2025/12 ~ 2026/04。

## 指標預覽

![8zz 反指標 - 元大台灣50 1小時線回測](assets/backtest-0050.jpg)

> **關於勝率的說明**
>
> 上圖在 0050（元大台灣50）上，固定8棒勝率最高約 **60%**。這不是指標不準，而是因為 8zz 喊的大多是**個別股票**（鈦昇、旺宏、白銀、記憶體股等），用 ETF 對照本來就會被稀釋。
>
> **用在她實際操作的個股上，勝率更高。**
>
> 指標顯示的方向如果有誤，那是她的錯，不是指標的錯。

## 如何安裝

> 如果你從來沒用過 TradingView，按照以下步驟一步一步來，大約 3 分鐘就裝好了。

### 第一步：開啟 TradingView 並登入

前往 [tradingview.com](https://www.tradingview.com)，免費註冊或登入帳號。
登入後點右上角「圖表」進入 K 線圖頁面。

![步驟1 - 開啟 TradingView 圖表頁面](assets/step-01-open-chart.png)

---

### 第二步：開啟 Pine Script 編輯器

在圖表頁面**最下方**找到「Pine Script 編輯器」標籤，點一下展開。

![步驟2 - 開啟 Pine Script 編輯器](assets/step-02-pine-editor.png)

---

### 第三步：複製指標程式碼

點開本頁面的 [`8zz-indicator.pine`](./8zz-indicator.pine)，按右上角複製按鈕（或全選 Ctrl+A → Ctrl+C）複製全部內容。

---

### 第四步：貼入編輯器並儲存

回到 TradingView，在 Pine Script 編輯器裡**清空原有的內容**，貼上剛才複製的程式碼。
然後點左上角「儲存」，取個名稱（例如：8zz反指標）。

![步驟4 - 貼上程式碼並儲存](assets/step-04-paste-save.png)

---

### 第五步：加到圖表

點「加到圖表」按鈕，指標就會出現在 K 線圖上了。

![步驟5 - 加到圖表](assets/step-05-add-to-chart.png)

---

### 第六步：設定可見性（重要）

箭頭是依據**精確時間**標記的，建議只在小時線以下使用。

右鍵點圖表上的指標名稱 → 「可見性」→ **取消勾選日、週、月**，只保留秒、分鐘、小時。

![步驟6 - 設定可見性](assets/step-06-visibility.png)

## 如何閱讀指標

### 箭頭

| 符號 | 顏色 | 意思 |
|------|------|------|
| ▲（K 棒下方） | 綠色 | 偏多 — 她停損/賣出了 |
| ▼（K 棒上方） | 紅色 | 偏空 — 她買進/加碼了 |

箭頭大小代表**訊號強度**：
- 大箭頭（★★★）— 停損出場、虧損X位數、all in 等情緒強烈用語
- 中箭頭（★★☆）— 買進、加碼、減碼、被套
- 小箭頭（★☆☆）— 觀望、持有、看多但未行動

### 懸停 Tooltip

把滑鼠移到箭頭上，會彈出她原始貼文的摘要，包含動作說明、操作標的、FB 發文時間。

![Tooltip 示意](assets/tooltip-example.png)

### 右上角資訊面板

| 欄位 | 說明 |
|------|------|
| 偏多 ▲ / 偏空 ▼ | 目前指標方向 |
| 倉位 | 當前訊號方向累積了幾格（最多 5/5 = 100%） |
| 訊號年齡 | 距最近一次翻轉多久了（超過48小時顯示「已過期」） |
| 翻轉次數 | 歷史上共翻轉幾次 |
| 固定N棒勝率 | 翻轉後持有固定根數的勝率 |
| 翻轉平倉勝率 | 持有到下一次翻轉才平倉的勝率 |
| 建議 | 哪種方式勝率更高 |

![資訊面板說明](assets/panel-example.png)

### 背景色帶

- **淡綠底**：目前指標偏多期間
- **淡紅底**：目前指標偏空期間

## 操作邏輯

```
1. 看到偏空 ▼ 出現  →  做空（或不做多）
2. 同方向每出現一次訊號  →  加倉 20%，最多加到 100%
3. 看到偏多 ▲ 出現  →  立刻全平 + 反手做多 20%
4. 訊號有效期約 1~3 天
5. 強度 ★★★ 的訊號，歷史上翻轉成功率更高
```

## 功能一覽

| 功能 | 說明 |
|------|------|
| **方向箭頭** | 方向翻轉時在 K 棒上/下顯示 ▲ / ▼ |
| **訊號強度** | 三段大小對應用語情緒強弱 |
| **背景色帶** | 偏多淡綠 / 偏空淡紅（可關閉） |
| **加倉箭頭** | 同方向重複訊號另顯示箭頭（預設關） |
| **Tooltip** | 懸停顯示原文摘要 |
| **資訊面板** | 方向、倉位、訊號年齡、雙模式勝率 |

## 參數設定

在圖表上右鍵指標名稱 → 「設定」可以調整。

### 顯示

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 箭頭距離 (ATR倍數) | `0.5` | 箭頭離 K 棒遠近，建議 0.3~0.8 |
| 顯示加倉箭頭 | 關 | 同方向訊號顯示額外箭頭 |
| 顯示背景色帶 | 開 | 偏多淡綠 / 偏空淡紅底色 |
| 顯示資訊面板 | 開 | 右上/右下統計面板 |
| 資訊面板位置 | 右上 | 可切換至右下 |

### 回測

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 固定觀察K棒數 | `5` | 翻轉後幾根 K 棒判定勝敗（1H線: 5=5小時；日線: 5=一週） |

## 這是什麼？

「8zz 反指標」是一個 TradingView K 線圖指標，忠實記錄了巴逆逆（8zz）在 Facebook 公開發文的每一筆買賣操作。

她的操作有一個神奇的規律：**買什麼跌什麼，賣什麼漲什麼。**

所以這個指標的邏輯也很簡單：

| 她做了什麼 | 指標顯示 | 你應該怎麼做 |
|------------|----------|-------------|
| 停損賣出 / 認賠畢業 | **偏多 ▲**（綠色） | 考慮買進 |
| 買進 / 加碼 / 抄底 | **偏空 ▼**（紅色） | 考慮賣出 |

收錄 **63 筆** FB 真實貼文操作，時間橫跨 2025/12 ~ 2026/04。每一筆都是她親自「奉獻」的市場數據。

## 安裝教學

> 從來沒用過 TradingView？按照以下步驟，大約 3 分鐘可以完成。

### 方法一：直接從 TradingView 頁面加入（推薦）

1. 開啟 [指標頁面](https://tw.tradingview.com/script/IbVmSqfj/)
2. 點「加到最愛指標」或「加到圖表」

就這樣。

---

### 方法二：手動複製程式碼

#### 第一步：開啟 TradingView 並登入

前往 [tradingview.com](https://www.tradingview.com)，免費註冊或登入。
登入後點右上角「圖表」進入 K 線圖。

![步驟1 - 開啟 TradingView 圖表](assets/step-01-open-chart.png)

---

#### 第二步：開啟 Pine Script 編輯器

圖表頁面**最下方**有「Pine Script 編輯器」標籤，點一下展開。

![步驟2 - 開啟 Pine Script 編輯器](assets/step-02-pine-editor.png)

---

#### 第三步：複製指標程式碼

開啟本 repo 的 [`8zz-indicator.pine`](./8zz-indicator.pine)，全選複製（Ctrl+A → Ctrl+C）。

---

#### 第四步：貼入並儲存

清空編輯器原有內容，貼上程式碼，點左上角「儲存」，取個名稱。

![步驟4 - 貼上程式碼並儲存](assets/step-04-paste-save.png)

---

#### 第五步：加到圖表

點「加到圖表」，指標出現了。

![步驟5 - 加到圖表](assets/step-05-add-to-chart.png)

---

#### 第六步：設定可見性（重要）

箭頭是根據精確時間標記，日線以上會對不上。

右鍵點指標名稱 → 「可見性」→ **取消勾選日、週、月**，只保留分鐘、小時。

![步驟6 - 設定可見性](assets/step-06-visibility.png)

## 如何閱讀指標

### 箭頭

| 符號 | 位置 | 意思 |
|------|------|------|
| ▲ 綠色 | K 棒下方 | 偏多 — 她停損/賣出了，該輪到她對了 |
| ▼ 紅色 | K 棒上方 | 偏空 — 她買進/加碼了，請自行評估後事 |

箭頭大小代表**訊號強度**，大白話版本：

| 大小 | 強度 | 她說了什麼 |
|------|------|----------|
| 大箭頭 | ★★★ | 她在哭了（停損、虧損X位數、all in） |
| 中箭頭 | ★★☆ | 她在行動（買進、加碼、被套） |
| 小箭頭 | ★☆☆ | 她在旁邊看（觀望、持有、「考慮」） |

---

### 懸停 Tooltip

把滑鼠移到箭頭上，會看到她的原話摘要，例如：

> 「你再跌我再買！74.90」
> 「壯烈犧牲」
> 「不停損 只攤平!」
> 「我不是冥燈！」

這些都是真實發文，未經修改。

![Tooltip 示意](assets/tooltip-example.png)

---

### 右上角資訊面板

| 欄位 | 說明 |
|------|------|
| 偏多 / 偏空 | 目前指標方向 |
| 倉位 | 累積了幾格訊號（最多 5/5 = 滿倉） |
| 訊號年齡 | 距最近翻轉多久（超過 48 小時顯示「已過期」，代表她可能又有新動作了） |
| 翻轉次數 | 她改變心意了幾次 |
| 固定N棒勝率 | 翻轉後持有固定根數的勝率 |
| 翻轉平倉勝率 | 持有到她下次反悔才平倉的勝率 |
| 建議 | 哪種方式勝率更高 |

![資訊面板說明](assets/panel-example.png)

## 操作邏輯

```
1. 偏空 ▼ 出現  →  她買了，考慮做空或保持觀望
2. 同方向每出現一次訊號  →  加倉 20%，最多到 100%
3. 偏多 ▲ 出現  →  她停損了，立刻全平 + 反手做多
4. 強度 ★★★ 的訊號代表她情緒特別激動，歷史上翻轉力道更強
5. 訊號有效期約 1~3 天
```

## 參數設定

右鍵指標名稱 → 「設定」可以調整。

### 顯示

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 箭頭距離 (ATR倍數) | `0.5` | 箭頭離 K 棒遠近，建議 0.3~0.8 |
| 顯示加倉箭頭 | 關 | 她同方向加碼時也顯示箭頭 |
| 顯示背景色帶 | 開 | 偏多淡綠 / 偏空淡紅底色 |
| 顯示資訊面板 | 開 | 右上角統計面板 |
| 資訊面板位置 | 右上 | 可切換至右下 |

### 回測

| 參數 | 預設值 | 說明 |
|------|--------|------|
| 固定觀察K棒數 | `5` | 翻轉後幾根 K 棒判定勝敗（建議 1H 線用 5~8） |

## FAQ

**Q: 這個指標真的有用嗎？**
A: 在她實際操作的個股上，固定8棒勝率可達 60% 以上。拿 0050 測試是因為方便，但 0050 不是她喜歡的口味。

**Q: 為什麼建議用小時線？**
A: 她的貼文精確到分鐘，日線以上的 K 棒太粗，訊號會對不到正確的棒上。

**Q: 資料多久更新一次？**
A: 目前為手動更新，她每次有新動作就會新增事件。

**Q: 巴逆逆知道這個指標嗎？**
A: 應該不知道。如果她知道了並且買進這個專案，我們就要開始做空它了。

**Q: 這是在嘲笑她嗎？**
A: 不，這是在**致敬**她。沒有她長期、公開、從不刪文的精神，就不會有這 63 筆寶貴的數據。她是用行動在教育市場的人。

**Q: 這算投資建議嗎？**
A: 不算。這是一個娛樂指標。真的。請不要拿來做任何財務決策。就算你真的賺錢了，也請告訴自己那是運氣。
